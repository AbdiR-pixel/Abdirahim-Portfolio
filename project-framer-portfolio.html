Web Design / Framer

Personal Portfolio Website

A personal portfolio website designed and developed in Framer — featuring a dark theme, smooth scroll animations, interactive project showcase, and a fully responsive layout. Built to showcase design and development work to potential clients.

Framer UI Design Responsive Animations
Visit Live Site

Project Overview

This portfolio was designed from scratch as a personal branding exercise. The goal was to create something that not only showcased my design work, but also demonstrated my ability to build interactive, performant websites using modern tools.

Framer was chosen for its unmatched combination of design flexibility and development power — allowing me to create smooth animations, interactive components, and a responsive layout without sacrificing design quality.

Design Decisions

The dark theme was a deliberate choice — it conveys sophistication, reduces eye strain, and allows the project imagery to stand out dramatically. The amber/golden accent color adds warmth and energy without compromising the premium feel.

Typography plays a starring role, with large italic serif headings creating visual drama alongside clean sans-serif body text. The combination signals both creative personality and professional clarity.

Key Sections

  • Hero Section — Animated portrait with hover state, large typographic hero, availability badge
  • Services Grid — Expandable service cards with detailed capability listings
  • Project Showcase — Filterable project grid with hover animations
  • Testimonials — Client reviews with star ratings and avatars
  • Blog Section — Design articles with category filtering
  • Contact Form — Multi-field inquiry form with service and pricing selection
